APP'S Functions:

1. The App Contains a map in which all the coordinates are by default marked.
2. There is an input present which responds to user's query and retrieves the names from the objects.
3. When search is performed the names(in the form of a button) appears after the input which when clicked, 
zooms in the map and changes the scale to 10 and marker is centered on the map.
4. Whenever input's length gets to zero after backspacing the map comes back to its default coordinates.
5. An Event is also applied on each marker, when clicked it does the same functionality for exclusivity.
